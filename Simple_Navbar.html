📘 JavaScript Breakdown (how the navbar works)

🔹 1. DOM selection – We grab the hamburger button, nav menu, all links, and create an overlay.

🔹 2. Toggle function (toggleMenu()) – Adds/removes .active class on menu & hamburger. Updates aria-expanded for accessibility. Also toggles the overlay visibility.

🔹 3. Close function (closeMenu()) – Removes active classes and overlay, used after clicking a link or tapping overlay.

🔹 4. Event listeners – Hamburger click toggles menu. Each nav link click closes menu automatically (great UX). Overlay click also closes menu.

🔹 5. Resize handler – If window width > 768px and menu is open, we reset classes to avoid stuck state.

🔹 6. Active link highlight – Uses scroll or simple current page detection (demo shows first link active).

✨ All transitions are CSS‑based (slide, opacity, rotation) → but JavaScript controls class toggles.
